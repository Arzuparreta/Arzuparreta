# Kitchen Clock

Expo (React Native) kitchen timer app with multiple timers, notifications, and haptics.

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [Expo CLI](https://docs.expo.dev/get-started/installation/) via `npx`

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

Then open in Expo Go, or use `npm run android` / `npm run ios` / `npm run web` with the appropriate toolchain.
