# Archived plans

Completed or superseded **execution** plans belong here so the [active plans index](../README.md) and agent context stay small. **Rationale** and durable constraints live in [`../../decisions/`](../../decisions/); full prose history remains in **git** for removed files.

When retiring a plan: move its `.md` into this directory (or delete it if nothing links to it anymore), run `make plans-index`, and ensure any non-obvious outcomes are captured in `docs/decisions/` if not already.
