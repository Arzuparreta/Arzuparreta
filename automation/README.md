# Automation

This area is for maintenance helpers used by you and coding agents.

- `scripts`: generated docs and maintenance utilities.
- `templates`: starter files and reusable patterns.
- `docs-kit`: shared docs snippets and writing helpers.
- `ci`: workflow support files.

## What is automated

- Project index generation from `projects/projects.json`.
- Internal link map generation across core sections.
- Workflow-based auto-refresh of generated knowledge pages.
